# Windows XP-Inspired Portfolio Website

A personal portfolio website inspired by the classic Windows XP desktop aesthetic.  
This project recreates retro UI elements in the browser while presenting my profile and work in an interactive format.

## ✨ Features

- 🖥️ Windows XP-style visual design and layout
- 🧩 Desktop-inspired UI components and interactions
- 📱 Responsive web layout for different screen sizes
- 🚀 Deployed with GitHub Pages

## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **GitHub Pages** (deployment)
